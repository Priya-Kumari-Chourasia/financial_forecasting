# 📈 Enhancing Neural Networks for Financial Forecasting

## Overview

Enhancing Neural Networks for Financial Forecasting is a deep learning-based stock price prediction system that uses multiple neural network architectures and technical indicators to forecast future stock prices. The application provides an interactive dashboard for stock analysis, model comparison, performance evaluation, and future price prediction.

## Features

* Real-time stock market data retrieval using Yahoo Finance
* Multiple neural network architectures:

  * Baseline MLP
  * Optimized MLP
  * Enhanced MLP
* Technical indicator-based feature engineering:

  * Moving Averages (MA)
  * Rolling Standard Deviation
  * RSI (Relative Strength Index)
  * MACD
  * Bollinger Bands
  * Log Returns
  * Volume Change
* Model performance comparison using RMSE, MAPE, and R² Score
* Early stopping and dropout regularization
* Interactive Streamlit dashboard
* Forecast visualization using Plotly

## Project Workflow

1. Collect historical stock market data using Yahoo Finance.
2. Perform feature engineering using technical indicators.
3. Preprocess and scale the data.
4. Train multiple neural network models.
5. Evaluate models using forecasting metrics.
6. Compare model performance.
7. Generate future stock price predictions.
8. Visualize results through an interactive dashboard.

---

## Model Architectures

### Baseline MLP

Input → Dense(64) → ReLU → Dense(1)

### Optimized MLP

Input → Dense(64) → ReLU → Dropout → Dense(32) → ReLU → Dense(1)

### Enhanced MLP

Input → Dense(128) → ReLU → Dropout → Dense(64) → ReLU → Dropout → Dense(1)

## Evaluation Metrics

The models are evaluated using:

* RMSE (Root Mean Squared Error)
* MAPE (Mean Absolute Percentage Error)

## Usage

1. Enter a stock ticker symbol (e.g., NVDA, TSLA, AMZN, HDFCBANK.NS).
2. Select the historical data period.
3. Choose the forecast horizon.
4. Select one or more neural network models.
5. Enable advanced technical indicators if required.
6. Click **Run Experiment**.
7. Analyze model performance and view forecasts.

---

## Results

The system compares multiple neural network architectures and identifies the best-performing model based on forecasting metrics. Predictions are displayed alongside historical stock trends and confidence intervals.
